# Register a new project

The first step into Tansu is to generate your project. After doing a search in the search bar, you will be prompted to register your project. Only 15 characters are allowed. The name must be unique on-chain.

:::warning[Registration collateral]

Only register projects you are a maintainer of and are allowed to register. Registration locks a **5 XLM collateral deposit**. Misuse and impersonation can be reported through governance and may result in forfeiture of that collateral.

:::

When registering a project, a `Tansu information file` is created for you in the background. This `tansu.toml` which is hosted on IPFS and holds metadata for your project. The dApp is using the data to make a nice project card.
